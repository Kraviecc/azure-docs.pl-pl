Zestaw SDK kotwic przestrzennych platformy Azure powinien zostać skonfigurowany w środowisku <a href="https://unity.com/releases/2019-lts" target="_blank">Unity 2019,4 (LTS)</a> przy użyciu następujących pakietów:
- AR Foundation: <a href="https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@3.1/manual/index.html" target="_blank">3.1.3</a>
- Rzeczywistość mieszana systemu Windows: <a href="https://docs.unity3d.com/Packages/com.unity.xr.windowsmr.metro@4.2/manual/index.html" target="_blank">4.2.1</a>
- Wtyczka ARCore XR (w przypadku obsługi urządzeń z systemem Android): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arcore@3.1/manual/index.html" target="_blank">3.1.3</a>
- Wtyczka ARKit XR (dla obsługi urządzeń z systemem iOS): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arkit@3.1/manual/index.html" target="_blank">3.1.3</a>
