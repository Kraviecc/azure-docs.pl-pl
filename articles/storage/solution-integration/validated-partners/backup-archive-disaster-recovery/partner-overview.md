# <a name="azure-storage-archive-backup-and-disaster-recovery-partners"></a>Archiwum, kopia zapasowa i partnerzy odzyskiwania po awarii usługi Azure Storage

W tym artykule przedstawiono partnerów firmy Microsoft zintegrowanych z usługą Azure Storage w celu archiwizacji i BCDR obciążeń. Wykorzystują one zalety skalowalności i [kosztów](https://docs.microsoft.com/azure/storage/blobs/storage-lifecycle-management-concepts?tabs=azure-portal) usługi Azure Storage, aby skutecznie rozwiązywać problemy z tworzeniem kopii zapasowych, tworzyć lokację odzyskiwania po awarii lub archiwizować nieużywaną zawartość do długoterminowego przechowywania. Wszystkie standardy zgodności spełniające wymagania usługi Azure Storage oraz [niezmienne Zarządzanie magazynem](https://docs.microsoft.com/azure/storage/blobs/storage-blob-immutable-storage) i cyklem życia umożliwiają łatwe zastępowanie kopii zapasowych opartych na taśmach i oferuje oszczędną witrynę odzyskiwania na żądanie.
